## ğŸ› Troubleshooting

### Photos Not Uploading
- Check file size (max 10MB)
- Verify format (JPG, PNG only)
- Ensure browser supports FileReader API

### Low Packing Efficiency
- Try MaxRects algorithm
- Reduce photo sizes
- Increase paper size
- Enable multi-page mode

### Export Issues
- Reduce DPI if PDF generation fails
- Close other tabs to free memory
- Export pages individually

### Cost Estimate Inaccurate
- Cost is estimated, not exact
- Based on average ink coverage
- Actual cost may vary by printer
